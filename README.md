# Política de Privacidade — Lembrei!

**Última atualização:** março de 2026

O aplicativo **Lembrei!** ("nós", "nosso") foi desenvolvido por Antonio Sergio da Silva Cezar Xavier. Esta página informa sobre nossas políticas de coleta, uso e divulgação de dados pessoais quando você usa nosso aplicativo.

---

## 1. Dados coletados

### Dados fornecidos pelo usuário
- **E-mail e senha** — utilizados para criação de conta e autenticação via Supabase.

### Dados gerados pelo uso
- **Notas, pastas, tarefas e destaques** — armazenados localmente no dispositivo e, para usuários Premium, sincronizados na nuvem via Supabase.

### Dados de compras
- **Informações de assinatura** — processadas pelo RevenueCat e Google Play. Não armazenamos dados de cartão de crédito.

---

## 2. Como usamos os dados

- Autenticar sua conta
- Sincronizar seus dados entre dispositivos (Premium)
- Processar assinaturas e pagamentos
- Enviar notificações locais de lembrete (sem envio de dados para servidores)

---

## 3. Compartilhamento de dados

Não vendemos nem compartilhamos seus dados pessoais com terceiros, exceto:

- **Supabase** — armazenamento e autenticação ([supabase.com/privacy](https://supabase.com/privacy))
- **RevenueCat** — gerenciamento de assinaturas ([revenuecat.com/privacy](https://www.revenuecat.com/privacy))
- **Google Play** — processamento de pagamentos ([policies.google.com/privacy](https://policies.google.com/privacy))

---

## 4. Armazenamento e segurança

- Dados locais ficam armazenados no seu dispositivo via AsyncStorage.
- Dados na nuvem são protegidos por autenticação e políticas de acesso (RLS) no Supabase.
- Não temos acesso ao conteúdo das suas notas sem sua autorização.

---

## 5. Retenção de dados

- Seus dados são mantidos enquanto sua conta estiver ativa.
- Você pode solicitar a exclusão da sua conta e dados a qualquer momento pelo e-mail abaixo.

---

## 6. Crianças

O Lembrei! não é direcionado a crianças menores de 13 anos e não coletamos intencionalmente dados de menores.

---

## 7. Notificações

O app envia notificações locais de lembrete de tarefas e destaques. Essas notificações são geradas no próprio dispositivo e não envolvem coleta ou envio de dados.

---

Você tem direito a:
- Acessar seus dados pessoais
- Corrigir informações incorretas
- Solicitar exclusão dos seus dados
- Portabilidade dos dados
